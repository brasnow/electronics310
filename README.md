# electronics310
Curriculum for a 4 unit 16 week junior level electronics course called Physics/Mechatronics 310 at California State University Channel Islands, 2025. 

More detailed outline was published in: 
Rasnow B (2024) Flipping the electronics lab: Learning upper division electronics at home. Amer. J. Phys. 92:809-818. https://doi.org/10.1119/5.020653, or https://www.rasnowpeak.com/brian/pub/flippedElab.pdf.
